# 🏋️ Fitness Tracking App (C Language)

A simple and interactive **menu-driven fitness tracking system** developed using C programming.
This application helps users monitor their daily health activities like steps, calories burned, BMI, and heart rate.

---

## 🎯 Purpose

This project is created to demonstrate the concept of a **real-time application** using C programming and to understand how fitness data can be processed and analyzed.

---

## 🚀 Features

* 👣 Step Counter
* 🔥 Calorie Calculation
* ❤️ Heart Rate Monitoring
* 📊 Fitness Report
* 🎯 Daily Goal Check (10,000 steps)
* ⚖️ BMI Calculator
* 🔄 Reset Data Option

---

## 📋 Menu Options

1. Enter Fitness Data
2. View Fitness Report
3. Check Daily Goal
4. BMI Calculator
5. Heart Rate Status
6. Reset Data
7. Exit

---

## ⚙️ How It Works

1. User enters fitness data (steps, weight, height, heart rate)
2. The system calculates calories burned
3. User can view reports and health status
4. Additional features like BMI and heart rate analysis are available

---

## 🛠️ Technologies Used

* C Programming Language
* Standard Input/Output Library (`stdio.h`)

---

## ▶️ How to Run the Program

### 🔹 Step 1: Compile the code

```bash
gcc fitness.c -o fitness
```

### 🔹 Step 2: Run the program

```bash
./fitness
```
